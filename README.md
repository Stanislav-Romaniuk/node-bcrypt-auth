# 🔐 Node Password Checker (bcrypt)

A simple CLI tool written in **Node.js** that securely stores and verifies a password using **bcrypt hashing**.

The script:

1. Accepts a password from the terminal
2. Saves the hashed password into a file (`password.txt`)
3. Compares future passwords with the stored hash
